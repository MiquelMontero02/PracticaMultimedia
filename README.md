# Fires i Festes WebPage 2.0

This webApp is focused on show more visibility on traditional Mallorca market along the months. This projects started as a final project of our degree subject called "Tecnologias Multimedia" and was focused in only be an static webPage that doesn't need a backend service.

In this new iteration, by my own, i'm going to implement a full-stack webApp that will
- Register all the events in a database
- Define a RESTFUL Api to interconenct the app with the real data
- Improve styles and performance of the app
